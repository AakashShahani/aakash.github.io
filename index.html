PROJECT IDEA

Smart Health Monitoring Application

Overview

The Health Monitoring application would focus on maintaining the health of people and advising them to achieve their goal. This would include diet monitoring, diet suggestions, footstep monitoring, sleep suggestion, water alarm. This application is for all the people who want to maintain their health without going to the gym. This app would be giving free trail for 1-2 months to all their user. In addition to that, Every-time user would be notified on their progress and would be rewarded with some discount coupons or free health checkup for being consistent on their goal.

h3> Description

The application would be available free for all the user who wants to keep monitor on their health whether he is young or old. Once the application gets installed on the device, he/she must have to register themselves with email and would have to enter their personal details, height, weight and their goal. After that, user must answer some basic questions about their health, if they might have any serious health issues. So accordingly, they would be provided their schedule. The user can see their profile anytime and can update his physical attributes which also effect on their routine. When the user opens the application, they would have to click on the start button to count their daily footsteps, calories in food, hours of sleep, how many glasses of water user drinks. And after observing few days of the user, system calculates the daily take of the user in 3 criteria’s: Proteins, Fats and Carbohydrates. The User must enter their daily intakes so that system can calculate how much he/she has consumed and how much he/she should take. The system has stored some food items calories, carbs, proteins in the database but the user can add new food items which will help the system to monitor the user diets. And the system would notify daily that how much the user had to walk in a day and how many calories they would have to take or burn depends on the user’s aim. For instance, if the user wants to gain some weight and he is willing to follow nutritional diet and he/she is not having some healthy grocery mentioned in the diet. So, the app can access device location and find him/her the nearby store where he/she can get the products. Every week the user can see his progress and status and for keeping them motivated they would get rewarded with some discount coupons or could get free health checkup at the prescribed stores. But this is only for the people who are serious about following all the prescribed activities time to time and are half way to their aim. Others gets some voucher for having started doing something for their health. The interesting fact about this application is that the user can sets alarm to get notified for something they want. Like the System tells the user how much water the user should intake and alarms him by notifying him about how much water he should have been consumed till now. They can set alarm for everything they want. System also suggest the user about hours of sleep he/she should take for the healthy life style. But if the user changes/updates something in their attributes, the output also changes accordingly. So, the user must enter correct details to get right output. And the user needs an active internet connection to use this application. User can have control over when they want to start their footstep counting, when to set alarm and when to get notified.

Tools and Technologies

As per my knowledge, android development tools like android studio, eclipse, game-maker studio etcetera is used and the device of windows 7/8/10 32-or-64bit with at least i3 processor or Mac OS X 10.10, minimum disk space of 3 GB, the memory of 3-GB ram and 1 GB for the android emulator and 1280 x 800 minimum screen resolution. These are the basic requirements for development of application. In addition to that, the visual studio or SQL server is needed and an android phone with KitKat version or higher than that.

Skills Required

The depth knowledge of the java programming language and to aware about all the software and hardware required for development of application. Able to work on the development tool like android studio, SQL server. Analytical mind and the high-quality user interface are the main factor while developing an application. And to think what the user wants while working on the application is the core factor to keep in mind. Creative mind is also needed to come up with something different and useful for the user.

Outcome

If the project gets succeed, then the people who want to track their health without going to the gym would be able to do it by just following some instructions from the application. The application would also help people to get their desired result.
